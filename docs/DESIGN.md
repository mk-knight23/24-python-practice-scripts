# Design System: 24-python-practice-scripts

## Visual Philosophy
The design follows a **Technical Laboratory / Education Hub** aesthetic. It uses a "Deep Lab" foundation (#0a0f1d) with signature "Python Blue/Yellow" accents, focusing on information density, hierarchical clarity, and encouraging technical persistence.

## Design Patterns
-   **Lab Sidebar**: Fixed, high-contrast navigation (P-8) with group-hover interaction and subtle internal gradients.
-   **Hud Cards**: Minimalist lesson containers (P-6) with 5% white opacity and emerald/python-colored accent categories.
-   **Glow Telemetry**: Circular progress indicators and streak modules with pulsing glow effects to Denote technical mastery.
-   **Syntax Containers**: Monospaced code blocks using emerald-400 highlights for a "Classic Terminal" output feel.

## Color Palette
-   **Lab Foundation**: Deep Navy/Black (#0a0f1d)
-   **Python Primary**: Azure Blue (Custom)
-   **Python Secondary**: Diagnostic Yellow (Custom)
-   **Success Emerald**: Test-passed indicator (#10b981)

## Interaction Design
-   **Sidebar Kinetic**: 400ms slide/fade reveals for navigation items to simulate a terminal HUD initialization.
-   **Lesson focus**: 1px border-python/50 transitions on card selection to maintain instructional context.
-   **Mastery Motion**: Dynamic SVG dash-offset transitions for real-time progress updates.
